# Retail_Sales_Analysis.
Project Overview

This project analyzes retail sales data using SQL to uncover business insights 
such as revenue trends, customer behavior, product performance, and sales distribution.

# KEY INSIGHTS
- Electronics category generated the highest revenue.
- Evening time (6 PM - 9 PM) had peak sales.
- Female customers contributed higher overall revenue.
- Revenue showed consistent weekly growth pattern.
